# home_server

This is an auth server for the homepage project I'm working on as an exploratory exercise.

It employs:
- MongoDB
- Mongoose (ORM)
- NodeJS
- ExpressJS
- morgan
- body-parser
- nodemon
- passport(jwt)

learning materials in the construction of this project.
